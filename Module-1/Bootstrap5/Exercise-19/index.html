Exercise 19: Customization with Sass

📌 How Sass Customization Works:
  1. Install Bootstrap via npm: npm install bootstrap
  2. Create a custom.scss file
  3. Override variables before importing Bootstrap:
    $primary: #e74c3c;
    $border-radius: 1.5rem;
    @import "bootstrap/scss/bootstrap";
  4. Compile Sass: npx sass custom.scss custom.css
  5. Link the compiled custom.css in your HTML
Result — Custom Primary Color (#e74c3c) & Border Radius (1.5rem)
