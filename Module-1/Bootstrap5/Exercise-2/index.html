Bootstrap 5 — Structure & Files

📁 css/

Contains compiled CSS files:

  • bootstrap.css — full unminified
  • bootstrap.min.css — minified (use in production)
  • bootstrap-grid.css — grid only
  • bootstrap-utilities.css — utilities only
📁 js/

Contains JavaScript plugin files:

  • bootstrap.bundle.min.js — JS + Popper bundled ✅
  • bootstrap.min.js — JS only (needs Popper separately)
  • bootstrap.esm.min.js — ES Module version
📁 icons/

Bootstrap Icons library:

  • 1800+ SVG icons
  • Use via <i class="bi bi-...">
  • Installable via npm or CDN
✅ Best Practice: Always use bootstrap.bundle.min.js — it includes Popper.js for dropdowns, tooltips, and modals.
